About

In this project I learned how to make a layout that was easily adaptable in font size and color scheme. I acheived this using a combination of useful new CSS features: variables, flexbox, and grid.

CSS Variables

I used variables to set colors and font sizes used consistently throuought a style, so that if I needed to change them, I could just change the variable. This made transitioning to a large, high-contrast layout very easy with a few simple variable changes.

CSS FlexBox

I used FlexBox to handle the navigation tabs. This allows the tabs to easily flow and flex as the size of the window and size of the tab content changes. When changing to a large font size, I did not need to modify the tab code because it simply flowed to fit.

CSS Grid

I used CSS Grid to layout the dial buttons. This made it very easy for me to define, in fractional units, that I needed 3 rows split 3 ways and a final row split two ways. This is often a feature that people have implemented in CSS frameworks in the past but is now a feature of CSS itself and is very easy to use! I also used it a bit in the nav.

Low Vision Mode (Large Mode)

For low vision mode I made sure that all the fonts were large and the colors of the doucment were very contrasted. I changed the hover color of the tabs too to add more contrast. It was nice that I never had to change the background button color between designs even though it looks like it changed. This was because I used a semi-transparent and dark button background color. So when the background of the document becomes darker, the button becomes darker. Lastly, I used a single HTML file and some vanilla javascript to toggle the low vision mode.
